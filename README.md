# AAAAAAAAAAAAAAAAAAAAAAAAAA

Крутая кроссплатформенная змейка на Python (`tkinter`) с:

- быстрым откликом на нажатия (буфер ввода + плавный игровой цикл);
- меню и экраном `Game Over` без закрытия окна;
- мгновенной переигровкой (`Enter`) и возвратом в меню (`Esc`);
- разными фруктами и встроенными pixel-art картинками (генерируются в коде);
- тремя бонусами:
  - `x2 очки` — удваивает очки за фрукты;
  - `Фантом` — можно проходить сквозь стены/себя (временный эффект);
  - `Слоумо` — замедляет игру на время;
- сохранением рекорда и истории игр в `snake_history.json`.

## Запуск

### Windows (PowerShell)

```powershell
py .\snake.py
```

### Linux/macOS

```bash
python3 snake.py
```

## Управление

- Стрелки / `WASD` — движение
- `Enter` — старт/переигровка
- `R` — рестарт прямо во время игры
- `Esc` — вернуться в меню

## Структура

- `snake.py` — вся игра в одном файле (включая спрайты фруктов и бонусы)
- `snake_history.json` — история игр и рекорды (создаётся автоматически)
