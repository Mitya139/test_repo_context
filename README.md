# test_repo_context

Минимальная кроссплатформенная змейка на Python (GUI на `tkinter`).

## Запуск

### Windows (PowerShell)

```powershell
py .\snake.py
```

Если команда `python` не работает, используй `py` (стандартный launcher на Windows).

### Linux/macOS

```bash
python3 snake.py
```

## Управление

- Стрелки или `WASD` — движение
- `Q` или `Esc` — выход
